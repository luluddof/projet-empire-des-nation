# Frontend Vue

## Prerequis

- Node.js 18+

## Installation

```bash
npm install
```

## Configuration

1. Copier `.env.example` vers `.env`.
2. Ajuster `VITE_API_BASE` si besoin.

## Lancer le frontend

```bash
npm run dev
```

Par defaut, le frontend demarre sur `http://localhost:5173`.

## Build production

```bash
npm run build
```
